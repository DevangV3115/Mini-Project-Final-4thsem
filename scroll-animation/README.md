# Mini Project - 4th Sem

**Learning Self-Correcting Reasoning in Large Language Models without Supervision**

## Project Structure

```
├── frontend/    ← Next.js web application
│   ├── src/
│   ├── public/
│   └── ...
├── backend/     ← Express.js API server
│   ├── src/
│   └── ...
└── README.md
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
